# Aman Patel — Developer Portfolio

A modern, animated developer portfolio website.

---

## 🚀 Quick Start

Just open `index.html` in your browser. No build tools needed.

---

## ✏️ How to Update Your Content

### Profile Photo
1. Place your photo at: `assets/profile.jpg`
2. Use a clean, professional headshot (square crop recommended)
3. If no photo exists, the section hides automatically

### Resume
1. Place your resume at: `assets/resume.pdf`
2. The "Download Resume" button links to this file

### Certificates
1. Place certificate images in: `assets/certs/`
2. In `index.html`, find the **Certifications** section
3. Uncomment the `<a>` tag and update the filename

### Personal Links
In `index.html`, search for and update:
- `your.email@example.com` → your actual email
- `https://github.com/` → your GitHub profile URL
- `https://linkedin.com/` → your LinkedIn URL
- `https://www.geeksforgeeks.org/` → your GFG profile URL
- `https://www.codechef.com/` → your CodeChef URL

### Adding More Projects
In `index.html`, find the commented template under `<!-- MORE PROJECTS -->` and copy it.
